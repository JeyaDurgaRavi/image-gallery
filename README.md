Project Structure
/src: Contains all the front-end source code (components, assets, styles).
/public: Publicly accessible files, such as images and the main HTML file.
/assets: Image files or media used in the gallery.
/styles: CSS or SASS files for the layout and design.
/components: React/Vue/Angular components (depending on framework).
index.html: The main HTML file.
package.json: Contains project metadata and dependencies.
